# Fahad Ahmed — Portfolio

A professional, animated portfolio website built with vanilla HTML, CSS, and JavaScript. Dark mode design with smooth transitions, interactive elements, and a complete showcase of skills, projects, and achievements.

## Features

- **Dark Mode Design** — Modern, sleek aesthetic
- **Smooth Animations** — Scroll reveals, hover effects, 3D transforms on project cards
- **Fully Responsive** — Works flawlessly on mobile, tablet, and desktop
- **No Dependencies** — Pure HTML/CSS/JS, lightning-fast load times
- **All Links Integrated** — GitHub, LinkedIn, coding profiles, certificates, and more
- **SEO Ready** — Proper meta tags, semantic HTML

## Sections

- Hero with animated stats
- About & Education
- Work Experience
- Featured Projects (PowerGrid AI, Walmart Platform, Algo University)
- Technical Skills (6 categories)
- Competitive Programming Achievements
- Coding Profiles (CodeChef, Codeforces, LeetCode, etc.)
- Leadership Roles
- Certifications with Coursera links
- Contact & Social Links

## Getting Started

1. Open `index.html` in any modern browser
2. All links and assets load from CDNs (no setup needed)
3. To deploy: Push to GitHub and connect to Vercel

## Deployment

Deploy to Vercel in 30 seconds:
1. Push this repo to GitHub
2. Go to vercel.com and import your GitHub repo
3. Done — your site is live!

## Customization

- Update links in the HTML (search for your GitHub URLs, emails, etc.)
- Change colors by modifying CSS variables at the top of `<style>`
- Add your resume PDF as `Fahad_Ahmed_Resume.pdf` in this folder

## Browser Support

- Mobile browsers (iOS Safari, Chrome Mobile)
